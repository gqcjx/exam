# 大嘴鸟 · 字词学习版

一个有趣的字词学习游戏，通过点击正确的字词来帮助大嘴鸟成长升级。

## 游戏介绍

大嘴鸟字词学习版是一款寓教于乐的学习游戏。玩家需要点击正确的字词，让大嘴鸟飞去"吞食"。通过不断学习，大嘴鸟会逐渐成长，解锁更多功能。

## 游戏规则

### 分数系统
- **点击正确词条**：+1分
- **点击错误词条**：-1分

### 升级系统
- **0级 → 1级**：累计18分，升级后重新计数
- **1级 → 2级**：累计36分，升级后重新计数
- **2级 → 3级**：累计72分，3级为最高级

### 特殊规则
- 每9个正确词必出1个错误词（用于学习纠错）
- 达到2级后，解锁"知新"功能按钮

## 操作说明

### 基本操作
- **点击字词**：大嘴鸟会飞向点击的词条并"吞食"
- **点击空白处**：大嘴鸟会飞向点击位置（不吞食）

### 快捷键
- **P键**：暂停/继续游戏
- **R键**：重置游戏

### 按钮功能
- **开始**：开始游戏
- **暂停**：暂停/继续游戏
- **重置**：重新开始游戏
- **音乐开关**：开启/关闭背景音乐
- **知新**：达到2级后解锁，跳转到学习报告页面

## 词库管理

### 默认词库
游戏内置了七年级期中版词库，包含常用字词及其拼音。

### 自定义词库
1. 点击"导入词库"按钮
2. 在文本框中输入词条，每行一个
3. 格式：
   - 正确词：`朗润`
   - 干扰项：`郎润/朗润`（错误词/正确词）
4. 可选择自动生成干扰项
5. 点击"确认导入"完成

## 技术说明

### 图片资源
- `kright.png`：大嘴鸟面向右侧的图片（向右、右上、右下移动时使用）

- 所有词条统一使用21px字号显示

- 低配模式自动检测
- 帧率自适应调整
