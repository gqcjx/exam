# Word 题库导入格式说明

本系统支持从 Word 文档（.docx）批量导入题目。请按照以下格式编写 Word 文档。

## 基本格式要求

### 1. 题目编号
题目必须以编号开头，支持以下格式：
- `1. 题目内容`
- `1、题目内容`
- `1) 题目内容`
- `第1题 题目内容`

### 2. 题干
题目编号后的内容即为题干，可以跨多行。

### 3. 选项（选择题）
选项必须以字母开头，支持以下格式：
- `A. 选项内容`
- `A、选项内容`
- `A) 选项内容`
- `A：选项内容`
- `A: 选项内容`

**注意：**
- 选项会自动识别为单选题或多选题（2个选项为单选，3个及以上为多选）
- 选项可以跨多行（续行会自动追加到上一个选项）

### 4. 答案
答案格式：
- `答案：A` 或 `答案: A`
- `正确答案：A,B` 或 `正确答案: A,B`
- 多个答案用逗号、顿号或空格分隔

**判断题答案：**
- `答案：正确` / `答案：错误`
- `答案：对` / `答案：错`
- `答案：T` / `答案：F`

### 5. 解析（可选）
解析格式：
- `解析：解析内容`
- `解析: 解析内容`
- 解析可以跨多行

### 6. 元数据（可选）
可以在题目中或文档开头设置以下元数据：

- **学科：** `学科：数学` 或 `科目：数学`
- **年级：** `年级：七年级`
- **难度：** `难度：2`（1-5）
- **标签：** `标签：几何,代数`（多个标签用逗号、顿号或空格分隔）

## 完整示例

### 示例 1：单选题

```
1. 下列哪一项是质数？
A. 21
B. 23
C. 25
D. 27
答案：B
解析：23 是质数，其余可分解质因数。
学科：数学
年级：七年级
难度：2
标签：数论
```

### 示例 2：多选题

```
2. 关于等腰三角形，哪些说法正确？
A. 两腰相等
B. 顶角平分线垂直平分底边
C. 底角相等
D. 三边都相等
答案：A,B,C
解析：等腰仅两腰相等，顶角平分线垂直平分底边。
学科：数学
年级：七年级
难度：3
标签：几何
```

### 示例 3：判断题

```
3. 地球绕太阳公转一周约 365.24 天。
答案：正确
解析：恒星年约 365.256 日，近似 365.24 天。
学科：科学
年级：七年级
难度：1
标签：天文
```

### 示例 4：填空题

```
4. 中国的首都是____。
答案：北京
解析：北京是中华人民共和国的首都。
学科：地理
年级：七年级
难度：1
```

### 示例 5：简答题

```
5. 简述光合作用的过程。
答案：光合作用是植物利用光能将二氧化碳和水转化为有机物和氧气的过程。
解析：光合作用包括光反应和暗反应两个阶段。
学科：生物
年级：八年级
难度：3
```

## 注意事项

1. **题目编号必须唯一**：每个题目必须以编号开头
2. **选项顺序**：选项按 A、B、C、D... 顺序排列
3. **答案格式**：答案必须明确标注，不能省略
4. **元数据位置**：元数据可以放在题目之前、之后或题目内部
5. **多题目分隔**：题目之间用空行分隔（可选，系统会自动识别）
6. **格式灵活性**：系统会自动识别多种格式变体，但建议保持格式一致

## 支持的题型

- **单选题** (`single`)：2个选项，单个答案
- **多选题** (`multiple`)：3个及以上选项，多个答案
- **判断题** (`true_false`)：正确/错误，答案 T/F
- **填空题** (`fill`)：题干包含 `____` 或 `______`
- **简答题** (`short`)：无选项，需要文字答案

## 导入步骤

1. 按照上述格式编写 Word 文档（.docx）
2. 在"题库管理"页面点击"导入题库（CSV/JSON/Word）"
3. 选择你的 Word 文档
4. 系统会自动解析并导入题目
5. 导入成功后，题目会出现在题库列表中

## 常见问题

**Q: 为什么我的题目没有被识别？**
A: 请检查：
- 题目是否以编号开头（1. / 1、/ 1) / 第1题）
- 选项格式是否正确（A. / A、/ A) / A：）
- 答案是否明确标注（答案：A）

**Q: 可以混合不同类型的题目吗？**
A: 可以，系统会自动识别题型。

**Q: 元数据是必须的吗？**
A: 不是必须的。如果没有设置，系统会使用默认值：
- 学科：未分类
- 年级：null
- 难度：1
- 标签：空数组

**Q: 支持图片吗？**
A: 目前不支持图片，只支持纯文本。

**Q: 可以导入多少题目？**
A: 理论上没有限制，但建议单次导入不超过 1000 题以保证性能。
